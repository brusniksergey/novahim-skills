# Скилл: Write-plan (план перед выполнением)

Перед запуском задачи подготовь:
- Цели (что должно получиться).
- Требования и ограничения.
- Зависимости (данные, доступы, люди).
- Этапы/майлстоуны.
- Ожидаемый результат и критерий готовности.

Верни короткий чек-лист. Для крупных задач — показать план и дождаться «ок» (правило Виктории).
